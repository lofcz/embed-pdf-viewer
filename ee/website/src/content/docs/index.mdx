# CloudPDF Documentation

Welcome to the CloudPDF documentation. CloudPDF is a cloud-based PDF viewer with
the platform features a real product needs — real-time collaboration, access
control, secure storage, forms, redaction, and e-signatures. It runs as a
managed service or self-hosted on your own infrastructure.
